# Project Chimera Skills

This folder contains the Skills that the AI agents can use.

A Skill is a reusable capability that an agent can call.

## Example Skills

### skill_fetch_trends

**Description:** Retrieves trending topics from social media platforms.

**Input:** 
- platform (e.g., Twitter, TikTok)
- date range

**Output:** 
- trend list with scores

---

### skill_generate_content

**Description:** Generates media content based on a trend.

**Input:** 
- trend topic
- content type (video, caption, hashtag)

**Output:** 
- media file path
- text content
